## 基础编辑

| 快捷键 | 模式 | 功能 | 说明 |
|--------|------|------|------|
| `h`, `j`, `k`, `l` | 普通 | 左/下/上/右移动光标 | 基础移动 |
| `w`, `b` | 普通 | 下一个/上一个单词开头 | 单词级移动 |
| `e`, `ge` | 普通 | 下一个/上一个单词末尾 | 单词级移动 |
| `0`, `$` | 普通 | 行首/行尾 | 行内移动 |
| `gg`, `G` | 普通 | 文件首/文件尾 | 文件级移动 |
| `i`, `a` | 普通 | 在光标前/后插入 | 进入插入模式 |
| `o`, `O` | 普通 | 在下方/上方新建行并插入 | 进入插入模式 |
| `dd` | 普通 | 删除当前行 | 删除操作 |
| `yy` | 普通 | 复制当前行 | 复制操作 |
| `p`, `P` | 普通 | 在光标后/前粘贴 | 粘贴操作 |
| `u` | 普通 | 撤销 | 撤销操作 |
| `Ctrl + r` | 普通 | 重做 | 重做操作 |
| `.` | 普通 | 重复上一次命令 | 重复操作 |
| `Ctrl + /` | 普通/可视 | 注释/取消注释 | LazyVim 默认注释功能 |
| `gcc` | 普通 | 注释/取消注释当前行 | LazyVim 默认注释功能 |
| `gc` + 动作 | 普通 | 注释/取消注释指定范围 | LazyVim 默认注释功能 |
| `>>`, `<<` | 普通 | 向右/向左缩进当前行 | 缩进操作 |
| `>`, `<` + 动作 | 普通 | 向右/向左缩进指定范围 | 缩进操作 |
| `Ctrl + u`, `Ctrl + d` | 普通 | 向上/向下滚动半屏 | 屏幕滚动 |
| `Ctrl + b`, `Ctrl + f` | 普通 | 向上/向下滚动一屏 | 屏幕滚动 |
| `zt`, `zz`, `zb` | 普通 | 将当前行置于屏幕顶/中/底部 | 视图调整 |

## 文件与缓冲区管理

| 快捷键 | 模式 | 功能 | 说明 |
|--------|------|------|------|
| `<leader>w` | 普通 | 保存当前文件 | 项目自定义 |
| `<leader>c` | 普通 | 关闭当前缓冲区 | 项目自定义 |
| `<leader>v` | 普通 | 关闭当前缓冲区 | 项目自定义 (使用 mini.bufremove) |
| `:w` | 命令 | 保存文件 | Neovim 原生命令 |
| `:q` | 命令 | 退出 | Neovim 原生命令 |
| `:wq` | 命令 | 保存并退出 | Neovim 原生命令 |
| `:q!` | 命令 | 强制退出不保存 | Neovim 原生命令 |
| `:e [文件名]` | 命令 | 打开/编辑文件 | Neovim 原生命令 |
| `<b`, `>b` | 普通 | 左移/右移当前缓冲区 | 项目自定义 (可带数字前缀) |
| `CpRelativePath` | 命令 | 复制当前文件的相对路径 | 项目自定义 |
| `CpFileName` | 命令 | 复制当前文件名(不带后缀) | 项目自定义 |
| `CpAbsolutePath` | 命令 | 复制当前文件的绝对路径 | 项目自定义 |
